# Resource-Documentation-and-Training

This repository contains documentation and training materials for HPC resources.

## Contents
- `docs/`: User guides and training notebooks.
- `scripts/`: Scripts for setting up environments.

## Requirements
- Python 3.x
- Conda

## Usage
1. Follow the instructions in `hpc_user_guide.md` to get started.
2. Use `setup_environment.sh` to set up your environment.
